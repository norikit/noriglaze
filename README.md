**noriglaze** is the theme manager for the **norikit** ecosystem. It stores your
themes, serves them to the other norikit tools, and switches the active theme
across all of them at once — so your tooling stays visually cohesive.

> [!NOTE]
> Work in progress. noriglaze is in early development and not yet usable.
